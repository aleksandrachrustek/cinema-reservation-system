\# Cinema Reservation System



Web application for cinema seat reservations.



\## Tech Stack

\- ASP.NET Core Web API

\- Entity Framework Core

\- SQLite

\- JavaScript



\## Features

\- User and guest reservations

\- Seat selection

\- Admin account

\- Demo data seed



\## Run project

1\. `dotnet run`

2\. Open browser at `http://localhost:5195/index.hmtl`
