# Cinema Reservation System

Web application for managing cinema seat reservations.

## Technologies
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- HTML, CSS, JavaScript

## Features
- User registration and login
- Browsing screenings
- Seat selection
- Seat reservation and cancellation
- Admin-ready architecture

## How to run
1. Open solution in Visual Studio
2. Run the project
3. Open http://localhost:5195/index.html
